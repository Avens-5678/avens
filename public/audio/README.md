# Audio Setup Instructions

To complete the background audio setup:

1. **Add your audio file**: Place your ambient/background music file at:
   ```
   public/audio/background-ambient.mp3
   ```

2. **Audio file requirements**:
   - Format: MP3 (recommended for browser compatibility)
   - Duration: 2-5 minutes (will loop automatically)
   - File size: Keep under 5MB for good performance
   - Type: Ambient, instrumental, or nature sounds work best

3. **Free audio sources**:
   - Freesound.org (Creative Commons licensed)
   - Zapsplat.com (free with registration)
   - YouTube Audio Library
   - Pixabay Music

4. **Controls**: Once the audio file is added, you'll see floating controls in the bottom-right corner of your website with:
   - Play/Pause button
   - Volume control (hover over volume icon)
   - Visual indicator when playing

The audio system respects browser autoplay policies - users will need to click play the first time to start the audio.
